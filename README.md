# Navabharath Technologies Website

This is a static HTML/CSS/JS website for Navabharath Technologies, designed based on the provided Figma mockup.

## Project Structure

- `index.html`: The main entry point containing all sections of the landing page.
- `css/style.css`: The main stylesheet containing all styles, variables, and responsive rules.
- `js/script.js`: JavaScript file for mobile menu interaction and form handling.
- `assets/`: Directory for images and icons.

## How to Run

Since this is a static website, you do not need any special server or installation.

1.  **Double-click** the `index.html` file to open it in your default web browser.
2.  Alternatively, you can use a local development server like Live Server (VS Code extension) or `python -m http.server` if you prefer.

## Features

- **Responsive Design**: Works on Mobile, Tablet, and Desktop.
- **Semantic HTML**: Uses proper HTML5 tags for better accessibility and SEO.
- **Clean CSS**: Uses CSS variables for easy theme management.
- **Interactive**: Includes a mobile navigation menu and smooth scrolling.

## Customization

- To change colors, edit the `:root` variables in `css/style.css`.
- To update content, edit the text within `index.html`.
